\# Visual Site Review



Après modification frontend :



1\. Lancer le site local.

2\. Vérifier desktop.

3\. Vérifier mobile 375px.

4\. Vérifier tablette 768px.

5\. Vérifier que le hero est compréhensible.

6\. Vérifier que le CTA est visible.

7\. Vérifier qu’aucune section n’est trop chargée.

8\. Vérifier que le site reste rapide et lisible.



Critère final :

le site doit être montrable à un commerçant sans explication orale.
