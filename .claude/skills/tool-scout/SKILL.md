\# Tool Scout



Tu dois chercher les outils existants avant de proposer d’en créer un nouveau.



\## Objectif



Aider à choisir les bons outils pour créer de meilleurs sites web avec Claude Code, Codex, Next.js, Tailwind, shadcn/ui, Supabase et GitHub.



\## Sources à consulter en priorité



1\. Claude Plugin Marketplace

2\. GitHub

3\. Documentation officielle des outils

4\. Context7 si disponible

5\. Annuaires MCP reconnus comme MCP.so ou Glama

6\. npm uniquement si le package est maintenu et documenté



\## Règle stricte



Ne jamais installer directement.

Ne jamais ajouter un MCP juste parce qu’il existe.

Toujours demander validation avant installation.



\## Critères d’évaluation



Pour chaque outil trouvé, vérifier :



\- utilité réelle pour Clic du Coin ;

\- maintenance récente ;

\- documentation ;

\- popularité raisonnable ;

\- permissions demandées ;

\- secrets/API keys nécessaires ;

\- redondance avec les outils déjà installés ;

\- risque sécurité ;

\- commande d’installation.



\## Outils déjà présents



\- GitHub MCP

\- Context7 MCP

\- Playwright MCP

\- Supabase MCP ou SDK Supabase

\- ui-ux-pro-max

\- Next.js

\- Tailwind

\- shadcn/ui

\- Framer Motion
